Overview

Purpose

The purpose of Living Words Today is to provide visitors with a simple, peaceful place to quickly find uplifting scripture passages and quotes from modern prophets and apostles. Each visit (or button click) shows a new quote with its reference and author, helping users get a small spiritual boost during busy days and discover verses or messages they might want to study more deeply.

Audience

The target audience includes:

  • Latter-day Saint students and young adults who want a quick spiritual pickup between classes or work.
  • Members who enjoy inspirational quotes but don’t want to scroll social media to find them.
  • Anyone who wants a clean, distraction-free page of scriptures and prophet quotes that look good on both phone and desktop.

Visitors are comfortable with basic web use (clicking buttons, navigating between pages) but don’t need any technical background.

Dynamic Elements (JavaScript)

  • Random quote on load: On page load, JavaScript selects a random quote object from an array and displays it in the main “Quote of the Day” card (scripture/quote text, reference, and author name plus image).
  • “New Quote” button: A button lets the user request another quote without reloading the page. An event listener on the button calls a function that picks a new random quote and updates the DOM.
  • Quote library filtering (child page): A separate Quote Library page lists all quotes from an array of objects. Dropdowns or buttons allow filtering by topic (Faith, Hope, Repentance, Charity) and/or author. Array methods like filter and forEach build the list of visible quotes. If no quotes match the filter, a message such as “No quotes found for this topic yet” is displayed.
  • Data structure & modules: Quotes are stored as objects in an array, with properties like text, reference, author, topic, and image. An ES module quotes.js exports the array, and other scripts import it. Multiple functions such as getRandomQuote(), displayQuote(), and filterQuotesByTopic() keep the code organized and show conditional branching.

Navigation

The main navigation will appear in the site header on every page:

  • Home – Random quote, featured quote card, and quick explanation.
  • Quote Library – Browsable/filterable list of all quotes and scriptures.

A third optional page (such as an About/Study Helps page) could be added if time permits, but the project will focus on these two required pages.

Content

Home Page – “Living Words Today”

  • Hero / Quote of the Day card: Dynamic quote area with scripture or quote text, reference, author name, and a small portrait image. Includes a “New Quote” button that triggers JavaScript to load another random quote.
  • How it works / About section: Brief explanation of the site’s purpose—quick spiritual boosts, curated quotes from scriptures and modern prophets—and a note about how topics and authors are tagged and used on the library page.
  • Topics preview: Small cards for a few key topics (Faith, Hope, Repentance, Charity) with one-sentence descriptions. Topic cards can link to the Quote Library page with a corresponding topic pre-selected.
  • Footer: Disclaimer that the site is a personal project and is not an official website of The Church of Jesus Christ of Latter-day Saints, plus links back to the top or to the Quote Library.

Quote Library Page

  • Page header: Title “Quote Library” and a short description: “Browse all scriptures and prophetic quotes in one place. Filter by topic or author.”
  • Filters bar: Dropdowns or buttons for Topic (All, Faith, Hope, Repentance, Charity, etc.) and Author (All, President Nelson, President Eyring, Elder Holland, etc.). These controls are wired to JavaScript to filter the list.
  • Quotes grid/list: Responsive grid or stacked list of quote cards generated by JavaScript. Each card includes quote text, scripture/talk reference, author name, and topic tags.
  • No-results message: When filters return zero quotes, show a message like “No quotes match this combination yet. Try another topic or author.”
  • Footer: Same footer as the home page for consistency.

Wireframes

Home Page Wireframe – “Living Words Today”

Top: full-width header with logo on the left and navigation links (“Home | Quote Library”) on the right. Below the header is a centered “Quote of the Day” card with text, reference, and author image, plus a “New Quote” button under the card. The next section shows a two-column layout on wider screens: left column with a short “How it works” explanation and right column with bullet points about the site’s purpose. A topics preview section displays 3–4 cards in a grid, each with a topic name and one-sentence description. The page ends with a footer bar that includes the disclaimer and a small link back to the top.

Quote Library Page Wireframe

Header is identical to the home page (logo and navigation). The top of the page shows the title “Quote Library” and a short description. Directly under that, a filter bar contains a topic dropdown and an optional author dropdown. The main content area is a responsive grid or column of quote cards generated by JavaScript, each displaying the quote text, reference, author, and topic tags. If no quotes match the current filters, a centered message appears below the filters. The page finishes with the same footer used on the home page.
